Pagination ka poora concept aur code aa gaya! Key points yaad rakho:

Formula ek hi hai:
OFFSET = (page - 1) * limit

Page 1 → skip 0, Page 2 → skip 10, Page 3 → skip 20 — bas itna hi hai pagination ka logic.

5 kaam controller mein hote hain:
Query params se page aur limit lo → OFFSET calculate karo → pehle COUNT(*) se total nikalo → phir LIMIT ? OFFSET ? se us page ke products fetch karo → response mein totalPages, hasNextPage, hasPrevPage bhi bhejo.

Demo mein pages click karke dekho — live LIMIT OFFSET values change hoti hain, aur products bhi change hote hain!

Bonus part important hai — real ecommerce mein category filter aur pagination dono ek saath kaam karte hain, woh code bhi widget mein hai. Agle step ke liye POST (product add karna) ya Search/Filter kaisa banate hain — jo chahiye bol do! 🚀
